# MTracker

A modern web-based financial tracking platform designed to help users organize expenses, monitor financial activity, and gain better visibility into personal budgeting and money management.

---

## Features

- Expense and income tracking
- Monthly financial overview
- Budget organization
- Financial data visualization
- Category management
- Responsive user interface
- User-friendly dashboard
- Real-time financial updates

---

## Tech Stack

- Ruby on Rails
- PostgreSQL
- JavaScript
- HTML & CSS
- Bootstrap
- Turbo Streams
- Git & GitHub

## Installation

Clone the repository:

```bash
git clone https://github.com/alessandracorreadev/mtracker.git
```

Install dependencies:

```bash
bundle install
```

Setup database:

```bash
rails db:create
rails db:migrate
rails db:seed
```

Start the server:

```bash
rails server
```
